# Onepage „Zrolowani" (M2) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Zbudować statyczny, przyciągający uwagę onepage promujący grę „Zrolowani" (Faza 1 z PRD), zgodnie ze specyfikacją `docs/superpowers/specs/2026-07-16-onepage-design.md`.

**Architecture:** Astro (SSG, statyczne prerenderowanie) + Tailwind 4. Strona złożona z małych, jednoodpowiedzialnościowych komponentów `.astro` scalanych w `index.astro`. Logika niewizualna (stan przycisku „KUP", metadane SEO) wydzielona do czystych modułów w `src/lib/` i testowana jednostkowo (Vitest). Komponenty testowane przez Astro Container API; całość weryfikowana buildem + testami E2E (Playwright).

**Tech Stack:** Astro 7, Tailwind CSS 4 (`@tailwindcss/vite`), Astro Fonts API (self-host Google Fonts), `astro:assets`, Vitest + `astro/container`, Playwright.

## Global Constraints

- Wersje bazowe (już w repo): astro `^7.1`, `@astrojs/cloudflare` `^14.1`, tailwindcss/@tailwindcss/vite `^4.3`. Menedżer: **pnpm** (`packageManager: pnpm@10.15.0`).
- Język strony: **polski**, `<html lang="pl">`.
- **Bez zewnętrznego CDN w runtime** — fonty self-hostowane przez Astro Fonts API; żadnych `<link>` do fonts.googleapis.com w wyprodukowanym HTML.
- **Minimum JS**: strona statyczna; JS tylko do sticky nav, menu mobilnego i animacji pojawiania (IntersectionObserver).
- **Jeden dominujący akcent: bursztyn `#F4B43F`.** Kolorowe obręcze (`#7B2CBF,#F4B43F,#FF5C7A,#33D17A,#35A7FF,#FF9F45`) **wyłącznie** przy 6 krokach „Jak grać".
- **Rytm tła:** sekcje 1–2 ciemne, 3–8 jasne (naprzemiennie biel `#FFFFFF` / lawenda `#F6F3FC`), 9–10 ciemne.
- Fonty: nagłówki/UI **Montserrat** (`--font-heading`), treść **Inter** (`--font-body`).
- **CTA „KUP"** sterowane jedną stałą `ALLEGRO_URL` w `src/lib/site.ts`. Pusty URL ⇒ przycisk w stanie „Wkrótce w sprzedaży" (nieaktywny).
- Każdy `<Image>`/`<img>` ma `alt`. Slogan-grafika: `alt="Prezydent, superbohater, a może kosmonauta?"`.
- Kontrast ≥ WCAG AA, widoczny focus, nawigacja klawiaturą, `prefers-reduced-motion` respektowane.
- Nazwy plików graficznych w repo: **ASCII** (bez polskich znaków).
- Każde zadanie kończy się zielonym `pnpm lint`, `pnpm format:check` (i testami zadania) oraz commitem.

- [ ] **Step 10: Zaimplementuj `src/lib/seo.ts`**

```ts
import { SITE_URL } from './site';

export interface Meta {
  title: string;
  description: string;
  canonical: string;
  ogImage: string;
}

const DEFAULT_TITLE = 'Zrolowani — imprezowa gra planszowa';
const DEFAULT_DESC =
  'Prezydent, superbohater, a może kosmonauta? Zrolowani to szybka, imprezowa gra, w której dopasowujesz znajomych do zwariowanych ról.';

export function buildMeta(o: {
  title?: string;
  description?: string;
  path?: string;
  ogImage?: string;
}): Meta {
  const title = o.title ? `${o.title} — Zrolowani` : DEFAULT_TITLE;
  const path = o.path ?? '/';
  const canonical = `${SITE_URL}${path === '/' ? '/' : path}`;
  return {
    title,
    description: o.description ?? DEFAULT_DESC,
    canonical,
    ogImage: o.ogImage ?? `${SITE_URL}/og.jpg`,
  };
}
```

```ts
export interface BuyState {
  enabled: boolean;
  href: string | null;
  label: string;
}

export function resolveBuy(url: string | null | undefined): BuyState {
  const clean = (url ?? '').trim();
  if (!clean) return { enabled: false, href: null, label: 'Wkrótce w sprzedaży' };
  return { enabled: true, href: clean, label: 'KUP' };
}
```

```ts
window.addEventListener('scroll', onScroll, { passive: true });

// Menu mobilne.
const toggle = document.querySelector('[data-nav-toggle]');
const menu = document.querySelector<HTMLElement>('[data-nav-menu]');
toggle?.addEventListener('click', () => {
  if (!menu) return;
  const open = menu.hasAttribute('hidden');
  if (open) menu.removeAttribute('hidden');
  else menu.setAttribute('hidden', '');
  toggle.setAttribute('aria-expanded', String(open));
});
menu?.querySelectorAll('[data-nav-link]').forEach((link) =>
  link.addEventListener('click', () => {
    menu.setAttribute('hidden', '');
    toggle?.setAttribute('aria-expanded', 'false');
  }),
);

// Animacja pojawiania sekcji (z poszanowaniem reduced-motion).
const reduce = window.matchMedia('(prefers-reduced-motion: reduce)').matches;
if (!reduce) {
  const io = new IntersectionObserver(
    (entries) => {
      for (const e of entries) {
        if (e.isIntersecting) {
          e.target.classList.add('is-in');
          io.unobserve(e.target);
        }
      }
    },
    { threshold: 0.12 },
  );
  document.querySelectorAll('[data-reveal]').forEach((el) => io.observe(el));
}
```
